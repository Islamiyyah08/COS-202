### Assignments

#### 1. Mathematical Calculator
A simple Python calculator that performs the following arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation
- Floor Division
- Clear
- Exit

#### 2. Personal Pocket CGPA Calculator
A Python program that calculates a student's CGPA by:
- Accepting course codes
- Accepting course units
- Accepting grades (A–F)
- Calculating quality points
- Calculating total credit units
- Displaying the final CGPA
- Displaying the class of degree
